# Responsive Image Gallery

This is a responsive web-based image gallery built with **HTML** and **CSS**, showcasing fashion items such as shoes, suits, and vintage wear. The project also includes an author section with contact details using Font Awesome icons.

## 📸 Features

- Stylish, responsive image grid layout
- Clean design with product names and links
- Author info section with profile photo, email, phone, and LinkedIn
- Font Awesome icons for better UX
- Fully mobile-responsive layout using Flexbox

## 🛠 Technologies Used

- HTML5
- CSS3 (Flexbox)
- Font Awesome (CDN)

## 🚀 Getting Started

To run this project locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/image-gallery.git

   ```

2. Navigate into the folder:
   cd image-gallery

3. Open index.html in any web browser.
   ✅ No build tools or installations required — it's fully static!
